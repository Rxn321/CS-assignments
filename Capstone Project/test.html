Codes i used in making this website

There are many different types of languages used for coding. The most popular ones are Java, Python, C++ and HTML. While the first three languages can be used in all types of applications, HTML(Hyper Text Markup Language) is the structure of all websites. Which is the main focus in my capstone project. Some of the most important html tags and elements i used in this website are:

html
: This is the most important tag of all, this tag tells the computer that this is a .html file, without this tag nothing would work.
Title
:This tag changes the website name shown on the users tab. This is important to show the user what kind of website they are on.
Link
:This tag is essential for anyone that wants to create a menu on their website or have a link.
Head
:Heading
Div
:div tag stands for “division” this tag usually is assigned to an element, which can then be styled and designed.
Style
:Without this tag a website would only have plain texts, this is necessary for having any designing in the website. Without it would probably look like this.
